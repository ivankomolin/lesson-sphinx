 - Docker v18.09+
 - Git
 - Make

## Clone repo
```
$ git clone https://github.com/ivankomolin/lesson-sphinx.git ./
```

1. Upload dump marvel.sql into exist database
2. Edit etc/sphinx.conf for setting exist database

## Example select single facet filter:
```
SELECT 
	GROUPBY() as value, 
	COUNT(*) as count 
FROM 
	marvel 
WHERE 
	MATCH('spider') AND eye!='' 
GROUP BY 
	eye 
ORDER BY 
	count DESC;
```
```
+-------+-------+
| value | count |
+-------+-------+
| Hazel |     1 |
| Red   |     1 |
| Brown |     1 |
+-------+-------+
3 rows in set (0.00 sec)
```

## Example select facet filter in the form of a slider:
```
SELECT 
	MIN(year), 
	MAX(year) 
FROM 
	marvel 
WHERE 
	MATCH('spider') AND year>0;
```
```
+-----------+-----------+
| min(year) | max(year) |
+-----------+-----------+
|      1954 |      2013 |
+-----------+-----------+
1 row in set (0.00 sec)

```
